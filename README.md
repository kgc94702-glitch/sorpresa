# Sorpresa romántica interactiva

Abre `index.html` en tu navegador para ver la sorpresa.

## Lo que incluye

- Animaciones de fondo, corazones flotantes y explosiones de partículas.
- Carta interactiva con texto que se escribe poco a poco.
- Sección de recuerdos interactiva.
- Cambios de atmosfera con transiciones y colores.
- Musica generada en el navegador.
- Opción para cargar una canción tuya con el botón `Cargar nuestra canción`.

## Cómo personalizarla rápido

- Cambia los textos principales en `index.html`.
- Cambia el mensaje de la carta en `script.js`, variable `letterMessage`.
- Cambia el mensaje final en `script.js`, variable `finalMessage`.
- Si quieres usar una canción especial, abre la página y usa el botón para cargar un archivo `.mp3` o similar.
